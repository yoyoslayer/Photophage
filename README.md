# Photophage. A pet that feeds on light

## What it is
I made a special virtual pet that you can carry around with you. It is like the Tamagotchi toys from the 90s but I built it from scratch. The pet is on a screen and it has buttons a buzzer, a light and a sensor that can tell if it is in the light or dark. All of this is inside an egg-shaped case that I made using a 3D printer.
What makes my virtual pet different is that it can tell what is going on in the world. It has a sensor that can tell if it is in sunlight, light or if it is dark. So it will react to where it's not just to what you do with the buttons.

## What it does / how to use it
- There are three buttons that you can use to take care of the pet: Feed, Play, Sleep.
- The pet has three things that you need to keep track of: Hunger, Happiness, Energy. These will go down over time so you need to check on the pet
If you press Feed, the pets Hunger will go up.
If you press Play, the pets Happiness will go up. Its Energy will go down a little bit.
If you press Sleep the pets Energy will go up.
- The pets face will change depending on how it's feeling. It will be happy if it is doing well okay if it is getting by and sad if something is wrong.
- There is a light that will change color depending on how the pet is feeling: green means it is happy amber means it is okay and red means it needs help.
- When you press a button the pet will make a beep.
- The pet can tell if it is in sunlight or indoor light and it will show you this every minutes. If it is dark it will not say anything.
- The pet runs on a battery that you can recharge and it lives in a custom egg-shaped case that I made.

## Why I made it
I made Photophage because I wanted to try something challenging. This was my time building a whole project from scratch and I wanted to add some special features like the light sensor and the mood light. I chose to make it a virtual pet that feeds on light because I think it is a fun and interesting idea. I used this project to learn about all the parts of building something, like this from designing the circuit board to writing the code and making the case.
